# For Dev

## Clone repo

```bash
git clone https://github.com/ForeverWeLearn/touchless.git
```

## Install packages

```bash
npm install
```

## Test

```bash
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

**Portable:** `./src-tauri/target/release/touchless.exe`  
**Installers:** `./src-tauri/target/release/bundle/`
